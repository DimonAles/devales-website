Horror Clicker Bosses (Unity)

A long-term experimental project that tracked my transition from mobile-based visual coding to professional game development in Unity.

View on Game Jolt

Development Evolution

This project is a historical marker in my coding journey, representing two distinct phases of my technical growth.

  • Phase 1: Pocket Code. The original concept was built on mobile using visual logic blocks. It allowed for quick prototyping of the clicker loop directly on a handheld device.
  • Phase 2: Unity Transition. Four years ago, the project was migrated to Unity. This required rewriting the entire logic in C# and adopting a professional component-based architecture.
  • Tech Stack: C# (Unity), Visual Logic (Pocket Code), 2D Sprite Rendering.
  • Character: Featured a classic Slenderman sprite as the interactive "boss".

Project Retrospective

Technical Leap

Migrating from Pocket Code to Unity was a major milestone, teaching me the fundamentals of the C# language and the complexities of a professional game engine.

Minimalist Design

The prototype remained focused on the core clicking interaction without audio, serving as a clean test-bed for UI/UX responsive logic in Unity.

Legacy Status

While the project is currently archived due to a shift in creative focus, it remains a testament to the beginning of my journey into advanced software development.
