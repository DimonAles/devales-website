YT Sub Counter (Python)

A desktop GUI application built with Tkinter for real-time tracking of YouTube subscriber counts using dynamic API polling.

Architecture

The application is structured as a standalone desktop tool that visualizes data retrieved from external sources. It moves away from command-line interactions to a user-friendly graphical interface.

  • GUI Framework: Built using Python's tkinter library with a custom canvas and responsive geometry management.
  • Data Fetching: Utilizes the requests library to consume third-party API endpoints (JSON) for channel searching and statistics.
  • Live Loop: Implements a non-blocking recursive update cycle (root.after) to refresh data every 5 seconds.
  • Search Logic: Automatically resolves channel names to IDs before fetching subscriber metrics.

Core Features

Visual Interface

Clean windowed GUI (`800x400`) providing clear visibility of metrics without needing a terminal.

Smart Search

Includes a name-to-ID resolver, allowing users to type a channel name instead of searching for complex Channel IDs.

Live Auto-Refresh

Automatically polls the API every 5 seconds to keep the subscriber count up-to-date in real-time.
