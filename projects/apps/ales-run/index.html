ALES RUN! (GDevelop)

A 3D endless runner designed for Android and PC. Navigate through obstacles, collect coins, and push your skills to the limit to beat the high score.

Play on Game Jolt

Development & Logic

This project was created as a learning milestone to master 3D game mechanics within the GDevelop engine. It focuses on lane-based movement logic and cross-platform input handling.

  • Engine: Developed using GDevelop, utilizing its 3D extension and event-based logic.
  • Input System: Implemented a dual-control scheme â€” Swipes for mobile devices and Keyboard (WASD/Arrows/Space) for PC.
  • Asset Integration: Used high-quality pre-made 3D assets to build a consistent and engaging visual environment.
  • Mobile Optimization: Logic and rendering were optimized to ensure smooth forward momentum on Android hardware.

Core Features

Responsive Controls

Fully optimized for mobile with swipe gestures and PC with keyboard support (Arrow keys, A/D for movement, W/Space for jumping).

3D Mechanics

Real-time 3D environment featuring procedural obstacle generation and lane-switching mechanics.

High Score Tracking

Integrated scoring system that records the player's distance and coin collection, encouraging repeated playthroughs.
