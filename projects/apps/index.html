Development Principles

  • Practical Problem-Solving: I focus on turning ideas into functional prototypes that solve real-world challenges, even on a small scale.
  • Iterative Development: Each application is built in small, manageable steps, which allows for frequent testing and validation of the core logic.
  • Technology Exploration: This is where I experiment with languages like Python and C++, and game engines like Unity, GDevelop, and Godot.
  • Clean Code: I prioritize writing readable and simple code, which is crucial for maintaining and scaling projects over time.
