My Development Workflow

I select the right stack for the job, building robust backends with Node.js and interactive frontends using JavaScript. All logic is structured for maintainability.

Projects are deployed on a variety of modern platforms including Vercel, Render, and GitHub Pages, while games are published on itch.io and Gamejolt.
