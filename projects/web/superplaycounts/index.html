SUPERPLAYCOUNTS

A comprehensive real-time analytics hub tracking social media metrics across YouTube, X (Twitter), and TikTok, powered by a custom API.

The Ecosystem

SUPERPLAYCOUNTS is more than just a counter; it's a Progressive Web App (PWA) that aggregates live data from multiple platforms into a single, unified interface.

Multi-Platform Tracking

Supports live stats for YouTube (Channels & Videos), Twitter/X (Followers), and TikTok (Beta), displaying data with smooth Odometer animations.

Interactive Tools

Includes a "Compare Mode" to view two channels side-by-side, a "Fake Counter" for content creation/pranks, and seasonal Countdowns.

User Experience

Features a responsive design with a global Dark/Light mode toggle, history tracking via updates log, and Disqus integration for community engagement.

Public API & Architecture

The project exposes a documented public API (`api.superplaycounts.pp.ua`) that allows other developers to consume the aggregated data.

  • Endpoints: Provides JSON data for Channel Search, User Statistics, and Video Metrics.
  • Search Logic: Custom search algorithms to find channels by name or ID.
  • Stability: Handles millions of requests (20M+ logged) with fallback mechanisms for API quotas.
  • Tech Stack: HTML5, CSS3, Vanilla JS, Service Workers (PWA), and a Node.js backend handling external API requests.
