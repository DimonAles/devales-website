TSP Team Music Site

A dedicated streaming portal built to showcase and promote music releases from the TSP Team YouTube channel, providing an ad-free listening experience.

Visit Platform

Technical Implementation

The project focuses on high-speed content delivery and a seamless user interface, acting as a direct extension of the YouTube promotional ecosystem.

  • Data Handling: Implemented a fast, lightweight JSON-based database system for immediate track loading and efficient metadata management.
  • Streaming Logic: Built with Node.js and Express, utilizing compression and caching middlewares to ensure minimal buffering during playback.
  • Responsive UI: A custom Single Page Application (SPA) feel created using EJS and Vanilla JavaScript for a fast, app-like navigation.
  • Adaptive Design: Features a dynamic color engine that extracts primary colors from album art to theme the interface in real-time.

The Challenge: Syncing the Audio Engine

Problem

Creating a unified playback state between the desktop bottom-bar player and the full-screen mobile player proved to be complex.

Solution

By studying deep documentation and other open-source implementations, I developed a custom event-driven synchronization logic that maintains track progress across all UI views.

Learning Outcome

This challenge significantly improved my ability to read technical documentation and manage complex application states in Vanilla JavaScript.

Roadmap

  • Migrate the backend infrastructure to Cloudflare Workers for global low-latency edge delivery.
  • Transition from flat JSON files to a high-performance SQL/NoSQL database for better data scalability.
  • Implement PWA (Progressive Web App) support for offline playback capabilities.
  • Develop a personalized "Recommended" algorithm based on listening history.
