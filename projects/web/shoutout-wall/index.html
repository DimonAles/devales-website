Shoutout Wall

A dynamic Node.js web application that displays a live, sorted leaderboard of YouTube channels using the YouTube Data API v3.

How It Works

The system is designed for streamers to allow viewers to "shout out" their channels on screen. It bridges a local Express server with the external YouTube API.

  • Ingestion: Users or bots trigger an endpoint (/api/id/:id) to add a channel ID.
  • Storage: The server maintains a rolling JSON database (FIFO) limited to the last 50 entries.
  • Fetching: The frontend periodically polls the API to fetch fresh avatars and subscriber counts.
  • Visualization: Channels are automatically sorted by subscriber count and animated using Odometer.js.

Key Features

Dynamic Sorting

The wall automatically re-orders cards in real-time. If a channel gains subscribers, it physically moves up the leaderboard.

Rolling Buffer (FIFO)

Logic implemented in Node.js ensures the file never grows too large, automatically removing the oldest entries when the limit (50) is reached.

Console Administration

Includes a CLI interface allowing the host to add channels directly from the server terminal using add <id> commands.
